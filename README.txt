PUBLICARE GRATUITĂ PE GITHUB PAGES
----------------------------------
- Creează un repository public pe GitHub.
- Încarcă TOATE fișierele și folderul assets în rădăcina repository-ului.
- Intră la Settings > Pages.
- La Source alege Deploy from a branch.
- Selectează branch-ul main și folderul /(root), apoi Save.

IMPORTANT DESPRE RSVP
---------------------
În această versiune, formularul și panoul admin folosesc localStorage.
Asta înseamnă că răspunsurile sunt vizibile doar în browserul în care au fost trimise.
Pentru utilizare reală cu invitații, formularul trebuie conectat la o bază de date online
(de exemplu Supabase sau Google Sheets prin Apps Script).

Fișiere principale:
- index.html — site public
- styles.css — design responsive
- app.js — countdown, animații și formular
- admin.html/admin.css/admin.js — panou administrare și export Excel CSV
- assets/ — fotografii
